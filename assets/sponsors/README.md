# Sponsor Logos

Drop your sponsor logo files here, named sequentially:

- `logo-01.png`
- `logo-02.png`
- `logo-03.png`
- ... and so on

Recommended format: PNG with transparent background, at least 200px wide.

The marquee in index.html references logo-01.png through logo-12.png by default.
Add more <img> tags in the .sponsors-track divs if you have more than 12 logos.
